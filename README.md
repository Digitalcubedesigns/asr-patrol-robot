## 📌 Project Overview

ASR is a fully autonomous patrol robot designed for perimeter security and surveillance. Built on the **AgileX LIMO Pro** platform using **ROS 2 Humble**, **LiDAR-based SLAM**, and **AI-powered vision** to detect, track, and alert on security threats in real time. Monitored remotely via a **Unity dashboard** over **LTE/5G**.

---

## ⚙️ Tech Stack

| Layer | Technology |
|---|---|
| Robot OS | ROS 2 Humble (Ubuntu 22.04) |
| Compute | NVIDIA Jetson AGX Orin |
| Robot Platform | AgileX LIMO Pro |
| Navigation | Nav2 + SLAM Toolbox |
| AI Vision | YOLOv8 + OpenCV |
| LiDAR | RPLIDAR A3 / Livox MID-360 |
| Cameras | PTZ IP Cameras + Intel RealSense D435i |
| Connectivity | LTE/5G (Teltonika RUT955) |
| Dashboard | Unity 3D + ROS-TCP Connector |
| Languages | Python 3.10, C++17 |
